this process handles the console pipelines of processes...
char mode pipeline should used... characters sent to this
process are displayed, words are sent when a key is pressed.
control characters are the following:

00h - beginning of command...
08h - go back one character...
0ah - go to next line...
0dh - go to beginning of line...

command characters are the following:
00h - no operation...
01h - report console resolution in a redraw, x, y sequence...
02h - clear the screen to the current color, cursor to upper left...
03h - set current color, the next byte contains the color...
04h - set cursor position, the next words are: x, y...
05h - write any special character...
06h - report capabilities in a noop; 0...
